## 3. TECH STACK (FULL‑STACK WITH BACKEND FOCUS)

### Backend

* **Runtime:** Node.js
* **Framework:** Express.js / NestJS
* **Language:** JavaScript (ES6+) / TypeScript
* **API Style:** RESTful APIs
* **Authentication:** JWT (Access & Refresh Tokens)
* **Validation:** Joi / Zod

### Database

* **Primary DB:** MongoDB / PostgreSQL
* **ORM / ODM:** Prisma / Mongoose / Sequelize
* **Caching (Optional):** Redis

### Frontend (Minimal – Portfolio UI)

* **Framework:** React.js (Vite or Next.js)
* **Styling:** Tailwind CSS
* **State Management:** React Hooks

### DevOps & Hosting

* **Backend Hosting:** Railway / Render / DigitalOcean
* **Frontend Hosting:** Vercel / Netlify
* **Database Hosting:** MongoDB Atlas / Supabase / Neon
* **Version Control:** Git & GitHub
